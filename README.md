Setup the library from another script
==========
. ${PATH_TO_SCRIP}/logger.ksh
LOG_PATH=${PATH_TO_LOG}
LOG_FILE=filename.log
LOG_LEVEL=0

Using the functions
==========
log_debug "Message"

log_error "Error message"
